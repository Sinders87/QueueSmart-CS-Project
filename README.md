QueueSmart is a queue management application designed to help users join and track service queues while enabling administrators to manage services and monitor usage.

This repository contains:

• Design documentation for Assignment 1 (A1), including system architecture, data model design, and role definitions.  

• Front-end UI/UX implementation for Assignment 2 (A2), including authentication flows, user dashboards, queue interaction screens, and mock data integration.  

• Back-end implementation for Assignment 3 (A3), including REST-style APIs, authentication logic, queue management, wait-time estimation, notifications, and in-memory data handling.  

• Data Design & Database Implementation for Assignment 4 (A4) adds database integration using a relational database, enabling persistent storage and retrieval of users, services, queues, and history through backend APIs.
